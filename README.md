## 🚀 快速开始

### 在线体验
直接访问 [draw.clare-ai.fun](https://draw.clare-ai.fun) 即可开始游戏，无需安装！

### 本地部署
```bash
# 克隆项目
git clone https://github.com/Clarexj/ai-draw-guess.git
cd ai-draw-guess

# 安装依赖
npm install

# 配置API密钥（可选）
cp .env.example .env
# 编辑 .env 文件，添加你的 SILICONFLOW_API_KEY

# 启动项目
npm run dev
```
